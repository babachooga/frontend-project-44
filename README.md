## Requirements

- Nodejs 22.x+ (required 100%)
- Make (not necessarily)

## Setup

```bash
make install
```

**OR**

```bash
npm run install
```

### Wellcome to the "brain games"!

[How to get greeting message?](https://asciinema.org/a/5ipbMgIGhiHgWYPj)

```bash
brain-games
```

### There are 5 types of games you play:

- **Brain calculation** [Can you solve the math problem ?](https://asciinema.org/a/u7yT18ddb64R6pRq)

```bash
brain-calc
```

- **Brain even** [Is it even number or not ?](https://asciinema.org/a/gSIjFzMVvPcHJHGv)

```bash
brain-even
```

- **Brain greatest common divisor** [What is the greatest common divisor ?](https://asciinema.org/a/s9KkzWPZQcHbUpAk)

```bash
brain-gcd
```

- **Brain prime** [Is it a prime number or not ?](https://asciinema.org/a/IcJYpY2MFCbsqWAY)

```bash
brain-prime
```

- **Brain progression** [What is the missing number in progression?](https://asciinema.org/a/sT8ngUinR2kO5css)

```bash
brain-progression
```
